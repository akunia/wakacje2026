PHOTOS (optionnel) — pour personnaliser les avatars
====================================================

Deux façons d'ajouter une photo :

1) DANS LA PAGE (par personne, privé)
   Ouvre index.html → bouton 🦸 → « Wgraj zdjęcie / Ajouter une photo ».
   La photo reste uniquement dans le navigateur de la personne (rien n'est publié).

2) « GRAVÉE » POUR TOUT LE MONDE (utile pour GitHub Pages)
   Dépose ici des images nommées exactement :
       paul.jpg
       tata.jpg
       katia.jpg
       alice.jpg
       wojtek.jpg
   Elles s'afficheront automatiquement comme avatars pour tous les visiteurs.

Priorité : photo uploadée dans la page  >  fichier photos/<id>.jpg  >  emoji.

Conseils : format carré, ~300x300 px, .jpg. (Le .jpg est attendu ; si tu as
un .png, renomme-le en .jpg ou convertis-le.)
